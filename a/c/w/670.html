Windows 系统通用快捷键

Win

打开「开始」菜单

Win + D

显示/隐藏桌面

Win + E

打开文件资源管理器

Win + L

锁定电脑或切换用户

Win + Tab

打开任务视图（多窗口切换）

Alt + Tab

快速切换当前打开的应用窗口

Alt + F4

关闭当前窗口或退出程序

Ctrl + Shift + Esc

打开任务管理器

文件资源管理器快捷键

Ctrl + N

新建文件夹

Ctrl + O

打开选中文件

Delete

删除选中文件（移至回收站）

Shift + Delete

永久删除文件（不经过回收站）

F2

重命名选中文件/文件夹

F5

刷新当前文件夹内容

Alt + 向左箭头

返回上一级目录

Alt + 向右箭头

前进到下一级目录

浏览器常用快捷键

Ctrl + T

新建标签页

Ctrl + W

关闭当前标签页

Ctrl + Tab

切换到下一个标签页

Ctrl + Shift + Tab

切换到上一个标签页

Ctrl + R F5

刷新当前网页

Ctrl + S

保存当前网页（图片/页面另存为）

Home

跳转到网页顶部

End

跳转到网页底部

Ctrl + I

斜体选中文字

Ctrl + U

Ctrl + 左/右箭头

在单词间快速移动光标（Word）

Ctrl + 上/下箭头

Ctrl + Shift + :

截图与录屏快捷键

PrtScn

Win + PrtScn

Alt + PrtScn

Win + Shift + S

打开系统自带截图工具（可自由截图、矩形/窗口/全屏截图）

Ctrl + Alt + A

截图

Ctrl + Alt + R

录屏
