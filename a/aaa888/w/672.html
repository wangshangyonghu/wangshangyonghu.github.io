WPS表格快捷键速查表

文件操作快捷键

新建表格 Ctrl + N
打开文件 Ctrl + O
保存文件 Ctrl + S
另存为 F12 或 Ctrl + Shift + S
打印 Ctrl + P

编辑操作快捷键

剪切 Ctrl + X
复制 Ctrl + C
粘贴 Ctrl + V
撤销 Ctrl + Z
重做 Ctrl + Y 或 Ctrl + Shift + Z
查找 Ctrl + F
替换 Ctrl + H
全选 Ctrl + A

单元格操作快捷键

插入行 选中行后按 Ctrl + Shift + +
插入列 选中列后按 Ctrl + Shift + +
删除行/列 选中行/列后按 Ctrl + -
合并单元格 Ctrl + M
取消合并单元格 Ctrl + Shift + M
移动单元格内容 选中单元格后，按Shift键拖动边框
复制单元格格式 Ctrl + Shift + C（复制）、Ctrl + Shift + V（粘贴）

格式设置快捷键

加粗文本 Ctrl + B
倾斜文本 Ctrl + I
下划线文本 Ctrl + U
设置单元格格式 Ctrl + 1
增加缩进 Ctrl + ]
减少缩进 Ctrl + [
应用货币格式 Ctrl + Shift + $
应用百分比格式 Ctrl + Shift + %

数据处理快捷键

自动求和 Alt + =
插入函数 Shift + F3
筛选数据 Ctrl + Shift + L
排序数据 Alt + D + S
选中当前区域 Ctrl + Shift + *
创建数据透视表 Alt + D + P

导航与选择快捷键

移动到表格边缘 Ctrl + 方向键
跳转到特定单元格 Ctrl + G 或 F5
选中连续单元格 Shift + 方向键
选中不连续单元格 Ctrl + 单击目标单元格

其他实用快捷键

切换工作表 Ctrl + PageUp（上一个）、Ctrl + PageDown（下一个）
插入工作表 Shift + F11
隐藏选中行/列 Ctrl + 9（隐藏行）、Ctrl + 0（隐藏列）
显示隐藏的行/列 Ctrl + Shift + 9（显示行）、Ctrl + Shift + 0（显示列）
快速计算选中区域 选中数值区域，状态栏显示求和、平均值等
