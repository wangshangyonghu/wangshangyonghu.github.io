三年级数学上册必背公式大全

一、时间单位换算

  • 1时 = 60分
  • 60分 = 1时
  • 1时 = 3600秒
  • 1分 = 60秒
  • 60秒 = 1分
  • 一刻钟 = 15分
  • 半小时 = 30分
  • 1天 = 24时
  • 1周 = 7天
  • 1年 = 4季度
  • 1世纪 = 100年
  • 1年 = 12月

二、时间计算方法

  • 开始时间 + 经过时间 = 结束时间
  • 结束时间 - 经过时间 = 开始时间
  • 结束时间 - 开始时间 = 经过时间

三、长度单位换算

  • 1米 = 10分米
  • 1分米 = 10厘米
  • 1厘米 = 10毫米
  • 10分米 = 1米
  • 10厘米 = 1分米
  • 10毫米 = 1厘米
  • 1米 = 100厘米
  • 100厘米 = 1米
  • 1分米 = 100毫米
  • 100毫米 = 1分米
  • 1千米 = 1000米
  • 1公里 = 1000米（注：千米和公里是同一单位）
  • 1000米 = 1千米
  • 1000米 = 1公里

四、面积单位换算

  • （进率为100）
  • 1平方米 = 100平方分米
  • 1平方分米 = 100平方厘米
  • 1平方米 = 10000平方厘米（100×100）
  • 1平方厘米 = 100平方毫米

五、质量单位换算

  • （进率为1000）
  • 1吨 = 1000千克
  • 1000千克 = 1吨
  • 1千克 = 1000克
  • 1000克 = 1千克

六、长方形与正方形周长公式

  • 长方形：周长 = (长 + 宽) × 2
  • 长方形：长 = 周长 ÷ 2 - 宽
  • 长方形：宽 = 周长 ÷ 2 - 长
  • 正方形：周长 = 边长 × 4
  • 正方形：边长 = 周长 ÷ 4
